# Ayaan → Tarana

An animated little proposal website from Ayaan to Tarana.

## Run locally

From this folder:

```bash
python3 -m http.server 5173
```

Then open:

- `http://localhost:5173`

## Add the background song

Put your song file here:

- `assets/ishq-bulaava.mp3`

Then open the site and press the **♪ Play** button on top.
